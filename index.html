Redmine diff mails

A Redmine plugin that adds colored diffs to wiki page change notification emails

Download this project as a .zip file Download this project as a tar.gz file

This plugin includes colored diffs in wiki change notification emails (so you don't have to click the diff link to see what's new).

Written by Boris Pilgun for http://www.gigantt.com and the rest of humanity.

To install just place this repository (redmine_diff_mails) inside the vendor/plugins folder in your Redmine installation and restart the web server.

Tested with Redmine 1.4.2.
